The indicators used by this website are based on data collected and owned by the local governments. This data is crucial to assess the preparedness of a city or province and to allow evidence-based decisions.

To maximise the impact of this data, it should open, which means accessible to third parties (citizens, NGOs, journalists...), in a easy to use format and a clear licence which allow analysis and transformation of the data, as outlined in the Open Definition. The openness of the key data used in this website vary a lot among the local governments of the Philippines. This index allows anyone to track which city or province opens their data and how open is it.
